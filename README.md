# Sistema Admnistrativo Lingo Lingo Kennel.

Este Projeto foi criado com [Angular](https://github.com/angular/angular) e [Electron.JS](https://github.com/electron/electron).

## Iniciando Scripts.

* Será necessário iniciar o BackEnd, [clique aqui para acessar](https://github.com/VitorCostaTI/LingoBack).

Você poderá ultiliza-lo em seu Browser, através do seguinte comando.

### `ng serve`

Abra [http://localhost:4200](http://localhost:4200) estará disponivel através desta porta local em seu Browser.

### `npm start` para abrir com App Local

Esta aplicação está em constante mudança então lembre-se de mante-la sempre atualizada e seguir as dicas aqui disponiveis.
